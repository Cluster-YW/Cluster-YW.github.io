// 前面加了两个斜杠，表示注释，大部分可能是我的碎碎念

涂黑的部分是一些隐藏的阴暗b设定，极有可能暗藏刀子或者涉及一些剧透，请谨慎食用。你知道的太多了（←当然左边这个没有）（来自萌娘百科的黑幕）

世界观设定

里界

// 脑子里好多东西，但是懒得写怎么办….

  • 平行于现实的另一个世界。
  • 所有的鬼都来自里界。

鬼

  • 拥有扭曲现实的力量，即使身处里界也能扭曲现实。
  • 当猎物（人）由于极度恐惧而失去理智时，可以直接将人拖入里界，之后发动物理攻击。
    所以鬼捕猎人的方式是这样的：
    在里界扭曲现实制造恐怖（鬼打墙、阈限空间、各种阴间场面）–> 把人拖入里界杀死 –> 吃掉。
  • “鬼是纯粹的欲望。”对于鬼来说，喜欢吃人是天然的欲望。鬼也不具有理性，但是鬼是有智慧的，也会使用一些计谋，不过仅限于满足自己吃人的欲望。
  • 鬼的弱点在于“目”。大部分的普通的鬼通体黑白，只有“目”呈现红色，如果可以在此处进行强力的打击，则可以一举直接杀死鬼。不过除非智力极其低下，一般来说鬼都会刻意把目藏在不容易被攻击的地方。

酒神与日神

  • 与人类共生的两位神。是某种概念系的存在，没有具体外貌。
    二神在人类诞生时一同出现，随着人类发展，日神日益强大，酒神力量衰弱，逐渐日神击败，并被驱逐到里界。但是20世纪以来，酒神已经开始恢复自己的力量。

主要人物

感谢 @独血 妈咪的赠图！！T T（左边是伍研，右边是安娜）

伍研

外观设定： 黑毛，黑瞳，单侧马尾。大概率穿着校服。一名普通的中国女高中生（嗯）。

你可以叫她阿研。

  • 正在和安娜同居。
  • 喜欢吃泡面。不喜欢吃除了鱼以外的海鲜。
  • 猎鬼人。到了夜间会出门猎鬼。
  • 睡眠不足，黑眼圈严重。因为晚上猎鬼回来还要补作业到深夜。不过学习成绩其实不错。
  • 能力是打开现实和里界来往的通道。
  • 患有严重的黑夜恐惧症。晚上睡觉不关灯，必须点一盏小夜灯。随身携带老师给的镇定药。原因是和小时候的好朋友误入里界导致朋友丧生，自己逃出生天。对黑暗的恐惧也是阿研最大的弱点。

安娜 Anna

外观设定 金毛，麻花辫。在学校穿校服。平时绝对不会穿，喜欢打扮自己。

在某次阿研猎鬼结束后从天而降砸到了阿研身上，于是某猎鬼女高包养了某流浪魔女（非自愿）。

  • 身世不明的魔女，目前正在与伍研同居（蹭吃蹭喝）。
  • 喜欢研究物理，常年物理成绩碾压阿研（让她很气）。
  • 喜欢在（伍研的）家里炼药/炼金。
  • 意外地对现代生活非常适应，甚至有自己的手机和社交账号。（从哪儿弄来的？？）
  • 目前世界上已知的唯一一位魔女。刻苦研究魔法的原理，想要借此搞清自己的身世。
  • 说是身世不明，其实已经失去了对过去的记忆，也不知道为什么自己拥有使用魔法的能力。事实上，和伍研的相遇时的从天而降，是因为在里界尝试从高处坠楼自杀。但是即将落地前看到了伍研于是用魔法急刹车。

莫诺 Mono

外观设定 由于是鬼所以不管什么形态都是红瞳。

  • 卧槽，鬼。
  • 可以变成猫，猫好。
  • 名字是伍研起的，现在住在伍研家里。（当成宠物养）（顺便看门）
  • 虽然身为鬼但是却有理性认知，可以在现实和人类社会中生存，相当特殊，也因此（仅仅）能够不被伍研直接灭了。

蠢蠢

  • 奇怪的人。
  • 强的不像话。
  • 活了多久啊这人我去。
